## Installation

**Clone repo:**
`git clone https://github.com/agent-seed/alarm-clock` or download zip and decompress it

**Enter app directory:**

`cd alarm-clock`

**Install required packages:**

`pip -r requirements.txt`

**Run:**

`python alarm.py`
